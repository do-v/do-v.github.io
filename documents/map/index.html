API BẢN ĐỒ

Cung cấp dịch vụ bản đồ số.

  • Tìm kiếm các đối tượng theo tên hoặc loại.
  • Tìm kiếm đối tượng theo vị trí.
  • Tuyến đường từ A đến B,...

Có thể tích hợp lập kế hoạch tuyến đường từ A đến B, điều hướng từng chặng, tính toán thời gian, để thực các nhu cầu cho khách hàng.

Được ứng dụng vào nhiều sản phẩm, nền tảng ứng dụng và giải pháp trên nhiều lĩnh vực về tối ưu hóa lộ trình vận chuyển, lập kế hoạch di chuyển, quản lý đội xe và nhiều hạng mục liên quan đến logistic và nhiều lĩnh vực khác.

Truy xuất API

Xác thực

Việc xác thực được thực hiện thông qua header HTTP.

<API_KEY>: Sử dụng khóa API đã được tạo.

curl -X GET \
-H "Authorization: ${API_KEY}" \
${API_URL}

Phần thân phản hồi là một đối tượng JSON chứa tất cả các trường API cung cấp.

Danh sách API

API URL HTTP Verb Chức năng
https://www.do-v.com/map_api?t=search&q=<name>&accept-language=vi GET Tìm kiếm các đối tượng theo tên.
https://www.do-v.com/map_api?t=reverse&lat=<latitude>&lon=<longitude>&accept-language=vi GET Tìm kiếm đối tượng theo vị trí.
https://www.do-v.com/map_api?t=route&point=<latitudeA>%2C<longitudeA>&point=<latitudeB>%2C<longitudeB>&locale=vi GET Tìm tuyến đường.

Kiểm tra API

  • Có thể sử dụng Postman request HTTP để kiểm tra
  • Giải mã hình nhiều đường được mã hóa ở route->path->points tham khảo tại đây
  • Có thể sử dụng TileMap tại đây
  • Đơn vị thơi gian sử dụng là mili giây
  • Đơn vị khoảng cách sử dụng là mét kiểu số chấm động
